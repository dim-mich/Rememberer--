# Rememberer--
проект для изучения платформы 1С:Предприятие 8.3

Для пакетного добавления данных Сервис - Сервис - Импорт данных из Табличного документа
Для изучения понятий (в примере это песни на английском языке и перевод построчно) нажать "Начать изучение понятий выбранной темы".

Структура данных в проекте:
Справочник понятий - это как изучаемое понятие, так и расшифровка этого понятия.
Справочник связей понятий - здесь понятия расшифровываются.
Темы - это иерархическое объединение подтем со связями понятий.

Обучение происходит путем самооценки праввильности ответа. В результате каждого ответа создается документ, с которым связан регистр накопления. Таким образом для каждой связи понятий формируется итоговая оценка изучения.
Оценка может быть с "-" или с "+" соответственно. Для новых связей оценка "0".
